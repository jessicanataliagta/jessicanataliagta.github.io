# Jessica Natalia GTA - GitHub Pages template

## Como usar

1. Extraia este ZIP.
2. Envie todos os arquivos para o repositório `jessicanataliagta.github.io`.
3. No GitHub Pages, use `main` e `/(root)`.
4. Acesse `https://jessicanataliagta.github.io`.

## Criar novo post

Copie uma pasta dentro de `posts/`, por exemplo:

`posts/proper-shaders-vc/`

Renomeie para:

`posts/nome-do-seu-mod/`

Edite o `index.html` dentro dela.

A URL ficará:

`https://jessicanataliagta.github.io/posts/nome-do-seu-mod/`

## Organização

- `assets/css/style.css` = aparência roxa do site.
- `assets/js/main.js` = pesquisa simples na home.
- `posts/` = páginas individuais.
- `mods/` = categorias por jogo.
- `tools/` = ferramentas.
- `categories/` = categorias gerais.
